To execute the project, run the applications in the following order:
1) server
2) client1 or client2

Executables are in /executables.

To run, open cmd in /executables and write 
-> java -jar server.jar
-> java -jar client1.jar
-> java -jar client2.jar

Every application is explained in the report:
- server runs the server;
- client2 populates the database;
- client1 retrieves the information from the server;
